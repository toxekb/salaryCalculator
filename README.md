"# salaryCalculator" 

**Requirements:**

PHP 7.4


**Installation:**

 `git clone https://github.com/toxekb/salaryCalculator.git`

 `cd salaryCalculator`

 `composer install`
 
 `doctrine:schema:create`
 
 `php bin/console doctrine:fixtures:load`
 
 **Using:**
 
 `php bin/console  calculate-salary`

**Tests:**

`php bin/phpunit`
